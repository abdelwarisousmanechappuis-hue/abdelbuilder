Gmail PDF Monitor — AbdelBuilder

Surveille tes mails, extrait les infos des PDF avec Google Gemini (gratuit), envoie via API.

1. Crée le projet Apps Script

Va sur script.google.com → Nouveau projet

2. Colle le code

Ouvre le fichier Code.gs, copie tout le contenu, colle-le dans l'éditeur.

3. Crée ta clé API Gemini (gratuite)

Va sur aistudio.google.com/app/apikey
Clique sur Créer une clé API → copie la clé.

Dans le code, remplace ta-cle-gemini-ici par ta clé.

Remplace aussi API_ENDPOINT par l'URL du logiciel du client.

4. Lance le déclencheur

Dans l'éditeur Apps Script :

⏰ Horloge → Ajouter déclencheur

Fonction : processPDFs → Temps → Toutes les 10 minutes

La 1ʳᵉ exécution demandera les autorisations (Gmail, Drive, URL Fetch). Accepte.

✅ C'est prêt !

Le script tourne 24h/24 gratuitement sur les serveurs Google.

Gemini 1.5 Flash = 0€ (60 requêtes/min gratuites).
